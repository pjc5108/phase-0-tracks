What are some common HTTP status codes?
	404 - Not found
	403 - Forbidden
	500 - Internal server error
	200 - OK
	301 - Moved Permanently


What is the difference between a GET request and a POST request? When might each be used?
Get is used to view information, while post is used when something needs to be changed, like a password or something.

Optional bonus question: What is a cookie (the technical kind, not the delicious kind)? How does it relate to HTTP requests?

A cookie is information that is stored locally on a users computer. It can be used to essentially save the state of a website, like remembering your username when you are logging in to a webpage.
